# 🚀 البدء السريع مع Barq Cashier

## الخيار 1: تشغيل الويب App (الأسرع والأسهل)

```bash
# 1. تثبيت المكتبات
npm install

# 2. بدء الخادم
npm start

# 3. افتح المتصفح على
http://localhost:3000
```

## الخيار 2: تشغيل تطبيق Desktop (.EXE)

```bash
# 1. تثبيت الحزم مع Electron
npm install

# 2. بناء التطبيق
npm run build-desktop

# 3. تشغيل التطبيق من مجلد dist/
# سيفتح التطبيق تلقائياً عند بناؤه
```

## 🛠️ استكشاف الأخطاء

### خطأ: Address already in use 3000
**الحل**: المنفذ 3000 مستخدم بالفعل
```bash
# اقتل العملية
taskkill /PID [PID] /F

# أو غيّر المنفذ في package.json
PORT=3001 npm start
```

### خطأ: Cannot find module
```bash
# أعد تثبيت المكتبات
rm -rf node_modules package-lock.json
npm install
```

### خطأ: Database is locked
```bash
# احذف ملف قاعدة البيانات وأعد بدء التطبيق
rm barq.db
npm start
```
